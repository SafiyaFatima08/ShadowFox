# 🦁 CSK Universe - The Ultimate Fan Hub 

A high-performance, interactive web portal dedicated to **Chennai Super Kings** fans. Built using **React.js** and **Vite** during the ShadowFox Internship, this project provides a premium experience to explore players stats, schedules, news, fanzone.

## ✨ Features

### 🦁 Core Functionality
* **Whistle Podu Hero Section:** Dynamic landing page with high-impact branding and "Explore Players" / "View Schedule" CTA buttons.
* **Dynamic Navigation:** Smooth transitions between Home, Players, Schedule, Stats, News, and Fan sections.
* **Player Hub:** Detailed statistics and performance metrics for the legendary CSK squad.

---

## 🛠️ Tech Stack
* **Frontend:** React.js (Functional Components & Hooks)
* **Build Tool:** Vite
* **Routing:** React Router DOM (v6)
* **Styling:** Modern CSS3 with CSK's Yellow & Blue theme
* **Hosting:** Vercel
